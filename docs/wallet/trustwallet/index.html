Using Trust Wallet for BNB Smart Chain

What is Trust Wallet?â€‹

Trust Wallet (trustwallet.com ) is a secure, open source, decentralized and anonymous Ethereum wallet application that supports Ethereum and over 20,000 different Ethereum based tokens (ERC20, ERC223 and ERC721), seeing significant user adoption since its launch in November 2017. The company has built a reputation for security and has held itself to the guiding principles that it will never access user wallets, hold private keys, and ask for personal information.

Install Trust Walletâ€‹

Go to https://trustwallet.com to install the most recent version of Trust Wallet.â€‹

Trust Wallet can be downloded from the Apple Appstore or from the Google PlayStore.

Launch the app in order to create a new wallet and accept the terms of use.â€‹

Backup the recovery phrase and keep it secure at all times.â€‹

Complete the initial setup and you're wallet is ready for use.â€‹

Getting the Smart Chain Addressâ€‹

Tap on the toggle at the upper right and then search for "Smart Chain".â€‹

Get the Smart Chain address by tapping on Copy, or Receive to display the QR code.â€‹

Receive BNB and BEP20 tokensâ€‹

Your Smart Chain address is able to receive BNB or BEP20 tokens.â€‹

Withdraw BNB Tokens from Binance.â€‹

BEP20 tokens are also supported by Trust Wallet.â€‹

Sending BNB to a Smart Chain addressâ€‹

Tap on Send to initiate the transfer, then fill in Recepient address and Amount.â€‹

Confirm the transaction by tapping on Send. Transaction will be done in seconds.â€‹

Access Binance Smart Chain DAppsâ€‹

Switching to Smart Chain Network to access DApps.â€‹

Type the URL on the built in browser in order to access a DApp.â€‹

Tap on the ETH logon on the upper right corner to see Network options. Choose Smart Chain.â€‹

Trust Wallet is fully compatible with Binance Smart Chain DApps.â€‹

More DApps will be added on the built-in browser for convenience.â€‹
