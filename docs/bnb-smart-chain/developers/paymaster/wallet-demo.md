# Experience Paymaster in mainstream Wallets

Several mainstream cryptocurrency wallets have already implemented Paymaster integration, 
while a significant number of other  wallets have yet to adopt this feature.

This tutorial will guide you through the experience of sending gasless transactions in both types of wallets.


## Paymaster Integrated Wallets

Wallets with integrated Paymaster functionality offer a seamless experience for users. 
These wallets automatically detect whether a transaction is eligible for sponsorship. 
When a transaction qualifies, the wallet sets the gas price to zero without any user intervention.

## Non Paymaster Integrated Wallets

Users of wallets without Paymaster integration can still benefit from gasless transactions through a manual process. 
By engaging with the community or participating in project-related activities, users can identify which transaction 
types are eligible for sponsorship. When initiating these qualified transactions, users can manually set the gas price 
to zero during the signing process. This proactive approach allows users to take advantage of zero gas fees, 
even in wallets lacking built-in Paymaster functionality.
